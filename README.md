# 用前准备

node 版本 14.18+ 或 16+ 或更高

全局安装

```bash
npm i szh-cli -g
```

npx 创建

```bash
npx szh-cli create <project-name>
```

查看命令详情

```bash
szh | szh --help | szh -h
```

查看版本

```bash
szh --version | szh -V
```

新建项目

```bash
szh create <project-name> -f (--force)
```

例如：`szh create vite-dmeo`

查看命令详情

```bash
szh <command> --help
```

例如：`szh create --help`
